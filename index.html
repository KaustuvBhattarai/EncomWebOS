»About

This is an experiment built by Kaustuv Bhattarai for simulating a web-based command line interface operating system.

Envisioned in 2020 with the concept of remotely overlooking the grid.

Automating the scheduling of rectifiers which are inconsitent on device time.

Using a time scale on the basis of a set of atomic clocks at the Monitor Stations and onboard the satellites. For within 1/15,000,000,000 of a second accuracy.

The OS also gives easy access to the Encom webframe and Database for remote administration.

[beta] The centralisation of the LLLSDLaserControl hub.


Depends on jQuery

License: MIT


Also makes slight use of OpenSUSE and CentOS for Server control

»Usage

sudo login -f (only for admin access)

login history (displays_encom_logins)

LLLSDLaserControl -ok (if_device_is_online) [no_of_apertures 1-2]

uname -a (displays_server_info) (temporary_invokes_admin)

whoami (displays_current_admin)

time (displays_system_time)

kill (terminates_process_instantly)


Disclaimer : Use with caution. Use of LLLSDLaser in Unready State may cause System Failure. Check [de-resolution] and [in-grid-desynchronization] before bootup.
